# 🚀 Deployment (GitHub Pages)

1. Create a repository named:

```
sachin-deepak-s.github.io
```

2. Upload your portfolio file:

```
index.html
```

3. Go to:

```
Settings → Pages
```

4. Select:

```
Deploy from branch → main → /root
```

5. Your site will be live at:

```
https://sachin-deepak-s.github.io
```
